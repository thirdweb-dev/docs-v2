# SendSwap

Send a quoted swap transaction. See [GetSwapQuote](/unity/pay/getswapquote) for more information on how to get a swap quote.

## Usage

```csharp
using Thirdweb.Pay;

public async void Buy()
{
	var txHash = await ThirdwebPay.SendSwap(_quote);
	ThirdwebDebug.Log($"Transaction hash: {_txHash}");
}
```

<Details summary="Configuration">

#### SwapQuoteResult

See [GetSwapQuote](/unity/pay/getswapquote) for more information on the `SwapQuoteResult` object.

### Return Value

`string`: The transaction hash of the swap transaction.
