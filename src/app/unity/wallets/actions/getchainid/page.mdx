# GetChainId

Get the chain ID of the currently connected wallet.

## Usage

Call this method to get the chain ID of the connected wallet.

```csharp
var chainId = await sdk.wallet.GetChainId();
```

## Return Value

Returns the chain ID as a `BigInteger`.

```csharp
BigInteger
```
