# ThirdwebProvider

`ThirdwebProvider` is a light-weight component that sets up React Query context for thirdweb SDK hooks.

```tsx
import { ThirdwebProvider } from "thirdweb/react";

function AppWithProvider() {
	return (
		<ThirdwebProvider>
			<App />
		</ThirdwebProvider>
	);
}
```
