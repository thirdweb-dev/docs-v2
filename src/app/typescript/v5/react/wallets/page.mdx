# Supported Wallets

These wallet config functions contains a built-in UI for connecting the wallet.

They cab be used with either
[`ConnectButton`](/typescript/v5/react/components/ConnectButton)
or [`ConnectEmbed`](/typescript/v5/react/components/ConnectEmbed)
components by passing it to the `wallets` prop

If you want to create a custom UI, You can use the underlying [wallet connectors](/typescript/v5/wallets) instead to connect a wallet, and set it as active wallet using the [`useConnect`](/references/typescript/v5/useConnect) hook

## Usage

```tsx
import { ThirdwebProvider } from "thirdweb/react";
import { ConnectButton } from "thirdweb/react";
// import the wallets you want to use
import { metamaskConfig, coinbaseConfig } from "thirdweb/react";

// pass them to the ConnectButton component
function App() {
	return (
		<div>
			<ConnectButton
				client={client}
				wallets={[metamaskConfig(), coinbaseConfig()]}
			/>
		</div>
	);
}
```
