# Supported Wallets

These wallet config functions contain a built-in UI for connecting the wallet.

They can be used with either
[`ConnectButton`](/typescript/v5/react/components/ConnectButton)
or [`ConnectEmbed`](/typescript/v5/react/components/ConnectEmbed)
components by passing it to the `wallets` prop

If you want to create a custom UI, You can use the underlying [wallet connectors](/typescript/v5/wallets) instead to connect a wallet, and set it as active wallet using the [`useConnect`](/references/typescript/v5/useConnect) hook

### Wallet connectors

The SDK supports 300+ wallets out of the box, all you need to pass is their id. Any injected or WalletConnect wallets can be accessed this way without any additional setup.

### Auto discovery of `EIP-6963` wallets

The SDK also supports EIP-6963 auto discovery, which means that it will automatically show any installed wallets that support this standard.

## Usage

```tsx
import { ThirdwebProvider } from "thirdweb/react";
import { ConnectButton } from "thirdweb/react";
import { createWallet, embeddedWallet } from "thirdweb/wallets";

// Choose which wallets to display
const wallets = [
	embeddedWallet(),
	createWallet("io.metamask"),
	createWallet("com.coinbase.wallet"),
	createWallet("me.rainbow"),
];

// Pass them to the ConnectButton component
function App() {
	return (
		<div>
			<ConnectButton client={client} wallets={wallets} />
		</div>
	);
}
```
