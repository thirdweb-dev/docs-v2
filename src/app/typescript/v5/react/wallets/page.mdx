# Wallets (React)

Adding these wallets in [`ThirdwebProvider`](/typescript/v5/react/ThirdwebProvider) enables connecting to these wallets in [`ConnectButton`](/typescript/v5/react/components/ConnectButton) and [`ConnectEmbed`](/typescript/v5/react/components/ConnectEmbed) components.

If you want to create a custom UI for connecting these wallets, Use the [Wallets (TypeScript)](/typescript/v5/wallets) instead.

## Usage

```tsx
import { ThirdwebProvider } from "thirdweb/react";
import { ConnectButton } from "thirdweb/react";

// import the wallets you want to use
import { metamaskConfig, coinbaseConfig } from "thirdweb/react";

function Example() {
	return (
		<ThirdwebProvider
			client={client}
			wallets={[metamaskConfig(), coinbaseConfig()]}
		>
			<App />
		</ThirdwebProvider>
	);
}

// render a ConnectButton or ConnectEmbed component anywhere in your app
function App() {
	return (
		<div>
			<ConnectButton />
		</div>
	);
}
```
