# AutoConnect

AutoConnect Last connected wallet user reloads the page or revisits the app at a later time.

<Callout variant='info'>
If you are using [`ConnectButton`](/typescript/v5/react/components/ConnectButton)
or [`ConnectEmbed`](/typescript/v5/react/components/ConnectEmbed) components, You don't need to use this component because it is already included.

</Callout>

### Example

```tsx
import { createWallet, embeddedWallet } from "thirdweb/wallets";
import { AutoConnect } from "thirdweb/react";

// wallets used in your app
const wallets = [
	embeddedWallet(),
	createWallet("io.metamask"),
	createWallet("com.coinbase.wallet"),
	createWallet("me.rainbow"),
];

function Example() {
	return (
		<AutoConnect
			client={client}
			autoConnect={{ timeout: 10000 }}
			wallets={wallets}
			appMetadata={appMetadata}
		/>
	);
}
```
