# Unified SDK (Beta)

<Callout title='Beta Release' variant='warning'>

You are currently viewing the documentation for the `Beta` version of the new thirdweb "unified" TypeScript SDK.

Refer to [stable version documentation](/typescript-sdks/) for production usage.

</Callout>

## Installation

```bash
npm install thirdweb@beta
```

## Requirements

- Node 18.6 or higher
- TypeScript 5.0.4 or higher (for TypeScript users)
- React 18.0.0 or higher (for React users)
