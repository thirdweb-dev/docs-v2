# TypeScript SDK (Beta)

Open source library to interact with any EVM-compatible blockchain.

<Callout title='Beta Release' variant='warning'>

You are currently viewing the documentation for the `Beta` version of the new thirdweb TypeScript SDK.

Refer to [stable version documentation](/typescript-sdks/) for production usage.

</Callout>

## Design Principles

- **Single package:** for Node, React and React Native with tree shaking for minimal footprint.
- **Performance optimized:** focuses on speed and efficiency with single responsibility principles.
- **Vertical integration:** direct access to all the thirdweb infrastructure from wallets to RPCs to IPFS.
- **Developer experience:** type safe, simple to use, with repeatable and predictable patterns.
- **Interoperable:** works with any infra provider and with other libraries like viem and ethers.

## Installation

```bash
npm install thirdweb@beta
```
