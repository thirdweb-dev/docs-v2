# Base Contracts

### ERC721

| Name                                                                | Description                                                                                                                                                                                                                                                                                 |
| ------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [ERC721Base](/solidity/base-contracts/erc721base)                   | Implementation of the [ERC721A](https://www.erc721a.org/) standard with the capability to mint NFTs.                                                                                                                                                                                        |
| [ERC721SignatureMint](/solidity/base-contracts/erc721signaturemint) | Adds [signature-based minting](/glossary/signature-based-minting) capability to the [ERC721Base](/solidity/base-contracts/erc721base) contract.                                                                                                                                             |
| [ERC721LazyMint](/solidity/base-contracts/erc721lazymint)           | Adds [lazy minting](/solidity/extensions/lazymint) capability to the [ERC721Base](/solidity/base-contracts/erc721base). Batch lazy-mint NFT metadata and have other wallets claim them under conditions you define in a `verifyClaim` function.                                             |
| [ERC721DelayedReveal](/solidity/base-contracts/erc721delayedreveal) | Adds [delayed reveal](/solidity/extensions/delayedreveal) to the [ERC721LazyMint](/solidity/base-contracts/erc721lazymint) contract.                                                                                                                                                        |
| [ERC721Drop](/solidity/base-contracts/erc721drop)                   | Adds [delayed reveal](/solidity/extensions/delayedreveal) and [claim conditions](/solidity/extensions/dropsinglephase) to the [ERC721LazyMint](/solidity/base-contracts/erc721lazymint) contract. Batch lazy-mint NFTs, and set up claim conditions to define how your NFTs can be claimed. |

### ERC1155

| Name                                                                  | Description                                                                                                                                                                                                                                                                                                                              |
| --------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [ERC1155Base](/solidity/base-contracts/erc1155base)                   | Implementation of the [ERC1155](https://eips.ethereum.org/EIPS/eip-1155) standard with the capability to mint and burn NFTs.                                                                                                                                                                                                             |
| [ERC1155SignatureMint](/solidity/base-contracts/erc1155signaturemint) | Adds [signature-based minting](/glossary/signature-based-minting) capability to the [ERC1155Base](/solidity/base-contracts/erc1155base) contract.                                                                                                                                                                                        |
| [ERC1155LazyMint](/solidity/base-contracts/erc1155lazymint)           | Adds [lazy minting](/solidity/extensions/lazymint) capability to the [ERC1155Base](/solidity/base-contracts/erc1155base) contract. Batch lazy-mint NFT metadata and have other wallets claim them under conditions you define in a `verifyClaim` function.                                                                               |
| [ERC1155DelayedReveal](/solidity/base-contracts/erc1155delayedreveal) | Adds [delayed reveal](/solidity/extensions/delayedreveal) to the [ERC1155Base](/solidity/base-contracts/erc1155base) contract.                                                                                                                                                                                                           |
| [ERC1155Drop](/solidity/base-contracts/erc1155drop)                   | Adds [delayed reveal](/solidity/extensions/delayedreveal) and [claim conditions](/solidity/extensions/dropsinglephase1155) to the [ERC1155Base](/solidity/base-contracts/erc1155base) contract. Batch lazy-mint NFTs, and set up [claim conditions](/solidity/extensions/erc1155dropsinglephase) to define how your NFTs can be claimed. |

### ERC20

| Name                                                                      | Description                                                                                                                                                                                                                         |
| ------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [ERC20Base](/solidity/base-contracts/erc20base)                           | Implementation of the [ERC20](/solidity/extensions/erc20) standard with support for the [EIP-2612](https://eips.ethereum.org/EIPS/eip-2612) standard.                                                                               |
| [ERC20Vote](/solidity/base-contracts/erc20vote)                           | Add [vote](https://github.com/thirdweb-dev/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/extensions/ERC20Votes.sol) capability to the [ERC20Base](/solidity/base-contracts/erc20base) contract.                    |
| [ERC20SignatureMint](/solidity/base-contracts/erc20signaturemint)         | Adds [signature-based minting](/glossary/signature-based-minting) capability to the [ERC20Base](/solidity/base-contracts/erc20base) contract.                                                                                       |
| [ERC20SignatureMintVote](/solidity/base-contracts/erc20signaturemintvote) | Adds [vote](https://github.com/thirdweb-dev/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/extensions/ERC20Votes.sol) capability to the [ERC20SignatureMint](/solidity/base-contracts/erc20signaturemint) contract. |
| [ERC20Drop](/solidity/base-contracts/erc20drop)                           | Adds [claim conditions](/solidity/extensions/dropsinglephase) capability to the [ERC20Base](/solidity/base-contracts/erc20base) contract. Distribute your token for a price under the criteria set up in claim conditions.          |
| [ERC20DropVote](/solidity/base-contracts/erc20dropvote)                   | Adds [vote](https://github.com/thirdweb-dev/contracts/blob/main/contracts/openzeppelin-presets/token/ERC20/extensions/ERC20Votes.sol) capability to the [ERC20Drop](/solidity/base-contracts/erc20drop) contract.                   |
