# Wallet Providers

## Modify displayed providers

To modify the displayed wallet providers, edit the `supportedWallets` array in `ThirdwebProvider`:

```jsx
import { ThirdwebProvider } from "@thirdweb-dev/react";

function App() {
  return (
    <ThirdWebProvider
      supportedWallets={[metamaskWallet(), walletConnect()]}
    >
      <ConnectWallet />
    </ThirdWebProvider>
  );
}
```

[View the full list of wallet providers.](/react/wallets)

## Recommend wallet provider

To display "Recommended" below a wallet provider, pass in the `recommended` property set to true:


```jsx 
import {
  ThirdwebProvider,
  ConnectWallet,
  metamaskWallet,
} from "@thirdweb-dev/react";

export default function App() {
  return (
    <ThirdwebProvider
      activeChain="<active_chain>"
      clientId="<your_client_id>"
      locale={en()}
      supportedWallets={[
        metamaskWallet({ recommended: true }),
      ]}
    >
      <ConnectWallet
        theme={"light"}
        modalSize={"wide"}
      />
    </ThirdwebProvider>
  );
}
```
