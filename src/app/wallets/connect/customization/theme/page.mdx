# Theme

## Default themes

To change the theme of the ConnectWallet component, pass in the `theme` prop with either the `light` or `dark` value.

Example:

```jsx  
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      theme="dark"
    />
  );
}
```


## Override default themes

To override the light or dark theme configurations, pass in the `darkTheme` or `lightTheme` functions as a base and override the corresponding values. 

Example:

```jsx
import { darkTheme, lightTheme } from "@thirdweb-dev/react";

<ConnectWallet
  theme={darkTheme({
    fontFamily: "Inter, sans-serif",
    colors: {
      modalBg: "#000000",
      accentText: "red",
      // ... etc
    },
  })}
/>;
```
