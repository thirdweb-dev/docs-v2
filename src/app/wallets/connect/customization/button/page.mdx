# Button

## Change disconnected state title

To change the displayed text when the connect button is disconnected, pass in a value to the `btnTitle` property.

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      btnTitle="Connect Wallet"
    />
  );
}
```

## Change button ui

To modify the default button UI, render a custom button and pass it into `detailsBtn`

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      detailsBtn={() => {
        return <button> .... </button>;
      }}
    />
  );
}
```
