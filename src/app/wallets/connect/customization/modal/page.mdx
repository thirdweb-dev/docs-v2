# Modal

## Change modal size

To modify the modal size, use the `modalSize` prop and pass in one of the following values:

- "wide", (recommended for desktop)
- "compact", (recommended for mobile)

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      modalSize="wide"
    />
  );
}
```

## Change modal title

To change the title of the Connect Modal title, use the `modalTitle` prop and pass in any string.

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
  return (
    <ConnectWallet
      modalTitle="Login"
    />
  );
}
```

## Welcome Screen

To customize the welcome screen on the modal, use `welcomeScreen` and modify the values. 

### Option 1 - Customize preset metadata

```jsx
<ConnectWallet
  welcomeScreen={{
    title: "Your Title",
    subtitle: "Your Subtitle",
    img: {
      src: "https://...",
      width: 300,
      height: 50,
    },
  }}
/>
```

### Option 2 - Render your own component

```jsx
<ConnectWallet
  welcomeScreen={() => {
    return <YourComponent />;
  }}
/>
