# Auth

Hooks provided to login, logout and get the user information. These hooks will use the domains setup in the `authConfig` defined in the [`ThirdwebProvider`](/react-native/v0/ThirdwebProvider).

| Hooks                                             | Description                                                   |
| ------------------------------------------------- | ------------------------------------------------------------- |
| [`useLogin`](/references/react-native/useLogin)   | Used to securely login to a backend with the connected wallet |
| [`useLogout`](/references/react-native/useLogout) | Used to logout the connected wallet from the backend.         |
| [`useUser`](/references/react-native/useUser)     | Used to get the currently logged in user.                     |
