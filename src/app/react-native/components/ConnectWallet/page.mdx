# ConnectWallet

ConnectWallet component renders a button which when clicked opens a modal to allow users to connect to wallets specified in the [ThirdwebProvider's supportedWallets](/react-native/connecting-wallets#supported-wallets) prop.

If `supportedWallets` is not configured in the [`ThirdwebProvider`](/references/react-native/ThirdwebProvider), the ConnectWallet Modal shows the below shown default wallets:

- [MetaMask](/react-native/wallets/metamask)
- [WalletConnect](/react-native/wallets/walletconnect)
- [Rainbow](/react-native/wallets/rainbowW)
- [Trust Wallet](/react-native/wallets/trust)

## Usage

```jsx
import { ConnectWallet } from "@thirdweb-dev/react-native";

function App() {
	return <ConnectWallet />;
}
```

We support localization of our SDK with support for a few languages.
You can pass a `locale` prop to our `ThirdwebProvider`, see more info in [Localizing our UI components](/react-native/ThirdwebProvider#localizing-our-ui-components).
