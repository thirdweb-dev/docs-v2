# Interact with wallets

Once connected, you can interact with the wallet using the `thirdweb/wallets/embedded` specific methods.

## Get the user email

```typescript
import { getAuthenticatedUser } from "thirdweb/wallets/embedded";

const user = await getAuthenticatedUser({ client });
```

## Transfer funds

```typescript
await embeddedWallet.transfer("0x...", 0.1); // amount in ETH
```

## Sign a message

```typescript

```
