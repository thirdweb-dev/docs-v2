# ConnectWallet

ConnectWallet component renders a button which when clicked opens a modal to allow users to connect to wallets specified in the [ThirdwebProvider's supportedWallets](/react/v4/ThirdwebProvider#supportedWallets) prop.

If `supportedWallets` is not configured in the `ThirdwebProvider`, the ConnectWallet Modal shows the below shown default wallets:

- [MetaMask](/references/react/v4/metamaskWallet)
- [Coinbase Wallet](/references/react/v4/coinbaseWallet)
- [WalletConnect](/references/react/v4/walletConnect)
- [Rainbow](/references/react/v4/rainbowWallet)
- [Trust Wallet](/references/react/v4/trustWallet)
- [Zerion Wallet](/references/react/v4/zerionWallet)
- [Phantom](/references/react/v4/phantomWallet)

## Usage

```jsx
import { ConnectWallet } from "@thirdweb-dev/react";

function App() {
	return <ConnectWallet />;
}
```
