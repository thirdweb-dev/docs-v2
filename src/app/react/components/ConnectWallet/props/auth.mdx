Enforce that users must sign in with their wallet using [auth](/wallets/auth) after connecting their wallet.

Requires the [`authConfig`](/react/react.thirdwebprovider#authconfig) prop to be set on the [`ThirdWebProvider`](/react/react.thirdwebprovider) component.

```jsx
<ConnectWallet
	auth={{
		loginOptional: false,
		onLogin(token) {
			console.log("user logged in", token);
		},
		onLogout() {
			console.log("user logged out");
		},
	}}
/>
```
