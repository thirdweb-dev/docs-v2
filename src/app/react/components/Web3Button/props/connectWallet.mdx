Web3Button renders a [ConnectWallet](/react/react.connectwallet) if no wallet is connected.

You can pass props for that `ConnectWallet` component by passing them in the `connectWallet` prop on `Web3Button`

```tsx
import { Web3Button } from "@thirdweb-dev/react";

<Web3Button
	connectWallet={{
		btnTitle: "Connect",
		modalTitle: "Login",
		// ... etc
	}}
	contractAddress="0x..."
	action={(contract) => console.log(contract)} // Logic to execute when clicked
>
	Execute Action
</Web3Button>;
```
